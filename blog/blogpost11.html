Belajar Vue.js

Saya cukup kewalahan ketika belajar vue, kalau boleh jujur framework ini buat sy mabok dengan semua atribut komponennya. banyak banget dan sy masih sangat awam untuk belajar bolak balik dari awal hingga membuat project dari vue semampu yang saya bisa buat.

Tapi ada satu hal yang sy rasa kemudahan menggunakan vue adalah ringkas dan teratur untuk kita menata sebuah project dalam keseluruhan. lalu banyak method baru dari vue yang menurut saya tidak ada di framework lain dan itu sangat memudahkan untuk mengerjakan beberapa real project.

Secara konsep dan penggunaanya saya sudah cukup paham walau masih harus banyak belajar lagi. project yang saya kerjakan pun cukup simple sekali dan belum ada kepuasan dalam diri melihat project tersebut. mungkin nanti dilain kesempatan saya bakal ngebuat project yang benar-benar serius pakai vuejs.
