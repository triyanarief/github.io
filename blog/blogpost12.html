Belajar Angular 2

Seperti postingan project saya yang sebelumnya saya mengerjakan sebuah web aplikasi sederhana untuk authentication user login.

Yang saya pakai bukan cuma angular, tapi MEAN stack yaitu MongoDb, Express, Angular, dan Nodejs. Selama proses pengerjaan saya cukup paham dengan basic dari node express dan mongodb karena pernah mengerjakan project marketplace.

Secara konsep dan penggunaanya saya sudah cukup paham walau masih harus banyak belajar lagi. Project kali ini sangat simple saja karena hanya menggunakan angular yang menurut saya salah satu framework yang mudah dipahami dibanding react.
