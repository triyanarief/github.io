Lolos Phase-0

Akhirnya tahap awal untuk masuk ke offline course phase-1 Hacktiv8 sudah kelar. Gak kerasa setelah sebulan mengerjakan tugas yang buat saya adalah hal baru semuanya.

Mulai dari week 1, saya belajar menggunakan HTML dan juga CSS untuk membuat tampilan HTML pertama saya yang bisa dibilang masih hancur banget hehehe. Tidak hanya itu kita juga belajar untuk membuat akun github dan belajar menggunakan command line dari git itu sendiri.

Masuk di week dua sampai dengan week empat kita mulai masuk pada fundamental programming. Belajar hal-hal mendasar tentang javascript mulai dari mendeklarasikan variabel sampai dengan belajar sintaks-sintaks dari EcmaScript terbaru.

Pada program phase-0 ini kita harus melewati live coding pada akhir pekan ke 2 sampai dengan pekan ke 4. Soalnya cukup menguras pikiran dan waktu sampai-sampai ada beberapa soal yang belum sempat dikerjain karena waktu yang sangat mepet. Tapi hasilnya cukup memuaskan karena rata-rata soalnya hampir mirip dengan materi yang kita kerjakan pada coderbyte di week ke 2 sampai dengan week ke 4. So, buat yang selalu takut untuk live coding jangan terlalu dianggap sebagai beban berat. karena pada intinya live coding sama halnya dengan ujian akhir semester yang bisa dibilang materinya dari yang pernah kita pelajari.

Satu hal yang saya sukai di-phase-0 adalah instruktur yang selalu aktif walau sudah diluar jam kerja mereka. Karena background saya yang non-IT jadi kebanyakan materi yang saya terima adalah hal yang benar-benar baru, jadi setiap ada soal atau materi yang kurang saya pahami saya pasti bertanya terus dan terus mungkin sampai instrukturnya bosen kali ya hehehe. Dan diakhir kita disuruh mendevelop aplikasi atau game sederhana dan saya cukup puas untuk membuat sebuah aplikasi sederhana yang gak bagus-bagus amat tapi lumayan untuk menambah nilai.

Misalnya salah satu pertanyaan konyol yang pernah saya tanyakan adalah saat koding saya tidak jalan. Saya panik nyoba ini itu tapi gak nemu masalahnya apa. Setelah dicek sama instrukturnya ternyata saya belum mendeklarasikan salah satu variabel didalamnya.

Kalau diingat-ingat memang membuat saya tertawa sendiri karena hal-hal seperti itu. Belajar ngoding buat saya berarti belajar untuk lebih teliti dan lebih logic lagi dalam menyelasaikan sebuah permasalahan.



Salam Terbaik.
