Learn Today Pt.4

Setelah kemarin saya belajar bootstrap maka saat ini saya masuk ke list berikut yaitu jquery. Memang rada ngaco alur belajarnya tapi gpp hanya untuk sekedar mengingat kembali apa saja yang kelupaan bisa menjadi bahan masukan kedepannya.

Sewaktu pelatihan kemarin saya sempat merasakan kelas yoga beberapa kali. nah pada saat itu saya kepikiran gimana nih iseng-iseng mau buat landing page untuk kelas yoga. keisengan itu akhirnya bisa terlaksana saat ini yaitu saya membuat dengan menambahkan jquery sederhana didalamnya yaitu carousel image pada header.

walaupun ini web masih sangat amat sederhana, tapi menurut saya masih mending dibanding dengan web bootstrap kemarin padahal disini saya hanya menggunakan HTML5 dan CSS3 plus jquery sebgai hidangan utamanya. Namun hasilnya cukup memuaskan buat saya. semoga kedepannya masih ada inspirasi yang datang untuk saya buat lagi web iseng-isengnya. hehehe...

Saya rasa itu saja yang ingin saya tulis sebagai pengingat dikemudian hari. Dan untuk yang mau tau hasilnya bisa Klik Disini
