Rencana Berikut

Dalam postingan sebelumnya saya sudah menyinggung tentang apa saja yang sudah saya pelajari dalam sepekan terakhir. dan saya lupa untuk menceritakan project kedua saya yaitu tentang Angular2 dan Firebase.

Jadi pada project ini saya membuat sebuah marketplace untuk orang-orang agar bisa berjualan dalam bidang properti. Saya bangun interface dengan Angular2 dan juga database menggunakan firebase realtime, lalu authentication user menggunakan google. Karena sebelumnya saya sudah mengerjakan project MEAN auth app dan juga marketplace untuk mac lovers, so, tidak ada kendala yang begitu berarti. dan saya dapat menyelesaikan dalam waktu 3 hari juga.

Rencana berikut yang ingin saya bahas disini adalah saya ingin melakukan pola hidup atau kebiasaan yang berkelanjutan untuk membiasakan diri tetap ngoding sehari minimal 2 jam atau lebih. dan juga minimal dalam seminggu saya harus terbiasa untuk mengerjakan project minimal 4 atau lebih.

Tidak hanya itu saya juga ingin melihat perubahan dalam diri saya tidak hanya melulu belajar pada hal yang sudah saya ketahui saja saya ingin terus berkembang kedepannya oleh karena itu sayang ini membuat beberapa komitmen untuk tetap belajar hal baru setidaknya dalam sebulan kedepan saya harus sudah bisa memahami end-to-end programming pada javascript.

Bagaimana untuk membangun web dan aplikasi dari awal, bagaimana untuk bisa mendeploy semuanya sampai live. dan juga kalau bisa saya harus bisa memahami konsep dan praktek fullstack maksimal 2 bulan kedepan. Ini adalah komitmen yang sudah saya pilih, apapun resiko dan masalah yang yang akan datang saya tetap akan menjalankan kebiasaan baik ini.

Jadi keinget salah satu kutipan "Kerja keras mengalahkan bakat!". dan saya akan buat review mingguan tentang apa saja yang sudah saya pelajari sebagai penanda saya sudah sampai sejauh mana dan sebagai bahan koreksi diri. Untuk yang mau tau project apa saja yang saya kerjakan kemaren bisa lihat di akun github saya ya. thanks!



Salam Terbaik.
